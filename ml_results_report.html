Model Descriptions

1. Persistence Model (Baseline)

Uses the temperature measured at 3 hours before sunset as the prediction. This simple baseline establishes the minimum performance threshold - any ML model should beat this.

2. Linear Regression

Fits a linear model using 16 engineered features including temporal patterns, statistical aggregates, and lagged temperatures. Features are standardized before training.

3. Random Forest

Ensemble of 200 decision trees that can capture non-linear relationships. Uses same features as linear regression but automatically learns feature interactions.

4. XGBoost

Gradient boosted trees with 200 estimators. Builds trees sequentially, with each tree correcting errors of previous ones. Often best performer for tabular time series data.
